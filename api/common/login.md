# Login - 登录

###### 应用场景

该接口提供用户验证，获取Token，并完成通过Token进行后续的业务逻辑。

###### 

###### 接口链接

> [http://{BaseURL}/OpenPlatform/Login](http://{BaseURL}/OpenPlatform/Login)

###### 

###### 请求参数

| 参数 | 说明 |
| :--- | :--- |
| account | 帐号 |
| pwd | 密码 |



###### 参数示例

> account=aaa&pwd=bbb



###### 返回结果

| 字段名 | 类型 | 说明 |
| :--- | :--- | :--- |
| code | String |  |
| msg | String |  |
| data | String |  |
